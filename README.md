### Setup Instructions

1. Clone the repository:
    ```bash
    git clone https://github.com/Alexiusss/habit-tracker -b HW_1
    cd habit-tracker
    ```
2. Build the project using Maven:
    ```bash
    mvn clean install
    ```
3. Run the application:
    ```bash
    java -jar ./target/*.jar
    ```
